Enhance the WooCommerce Stripe checkout with Rewardful's conversion tracking

== Installation ==

This plugin requires the [WooCommerce plugin](https://en-ca.wordpress.org/plugins/woocommerce/ "WooCommerce")  with WooCommerce Stripe activated.

1. Add `woocommerce-rewardful.zip` via Plugins > Add New
2. Activate the plugin through the 'Plugins' menu in WordPress
